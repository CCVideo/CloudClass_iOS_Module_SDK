◆ onStreamError:forStream:()

- (void CCStreamerBasicDelegate) onStreamError: (NSError *)  error
forStream: (CCStream *)  stream 
optional

Triggers when an error happened on a stream. This event only triggered for a stream that is being publishing or subscribing. SDK will not try to recovery the certain stream when this event is triggered. If you still need this stream, please re-publish or re-subscribe.

Parameters
streamThe stream which is corrupted. It might be a LocalStream or RemoteStream.
errorThe error happened. Currently, errors are reported by MCU.
